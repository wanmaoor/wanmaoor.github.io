em & rem in font-size
2018-10-03 10:35   folder FrontEnd label CSS

Today, I’m going to talk about font units em and rem, According to MDN , it said below that:

For most font-relative units (such as em and ex), the font size is relative to the parent element’s font size.

For font-relative units that are root-based (such as rem), the font size is relative to the size of the font used by the html (root) element.

I will show you the difference between two units with codes:

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>JS Bin</title>
</head>
<body>
  <p>我是文章</p>
  <div class="p1">
    <p>I'm a article</p>
  </div>
</body>
</html>

And then, we will define two CSS file to show their difference

/*1st*/

.p1{
  font-size: 40px;
}
.p1 p{
  font-size: 0.5em /*equal to 20px*/
}


/*2nd*/

.p1{
  font-size: 40px;
}
.p1 p{
  font-size: 2rem /*equal to 40px*/
}
